# Pomidor Timer #

A simple timer for use in <a href="http://pomodorotechnique.com">The Pomodoro Technique®</a>, a time management system. 

One of the great things about <a href="http://pomodorotechnique.com">The Pomodoro Technique®</a> is they provide copies of all the resources you'll need to use the system for free!  You can download the book and worksheets  

## Requirements ##

OS X 10.6

## Instructions ##

If you can't figure it out I can't help you... no one can.

## Why is this free? ##

They provide the resources for free so I thought I'd provide this simple timer for free. There are applications that cost money and do more but I wanted to do one thing and do it well; be a timer.
